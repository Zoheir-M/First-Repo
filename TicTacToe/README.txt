TIC TAC TOE

Tic Tac Toe is a simple web-based game created using HTML, CSS, and JavaScript. 
This game allows two players to take turns placing their marks (X and O) in a 3x3 grid. 
The goal is to be the first to get three of your marks in a rowâ€”horizontally, vertically, or diagonally.

- **Responsive Design**: The game is responsive and can be played on various devices.
- **Dark Theme**: A night-themed interface with rounded squares and a starry background.
- **Smooth Animations**: Clicks are animated for a smoother gameplay experience.
- **Turn Indicator**: Displays whose turn it is to play.
- **Winner Announcement**: Automatically announces the winner or declares a draw.

- **HTML**: For the structure of the game.
- **CSS**: For styling the game and creating a visually appealing layout.
- **JavaScript**: For game logic and interactivity.

1. Open the `tic_tac_toe.html` file in your web browser.
2. Players take turns clicking on empty squares to place their marks (X or O).
3. The game will announce the winner or declare a draw when the game ends.
4. Click the "Reset" button to start a new game.

To run this project locally:
1. Clone the repository or download the HTML file.
2. Open the `tic_tac_toe.html` file in any web browser.
3. Enjoy playing Tic Tac Toe!
